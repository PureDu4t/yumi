Jumpy Frog

Jumpy Frog

This game was made for my "Mixing For GAIMS" class at Berklee. I was tasked to create a walkable map that showcases basic implementation of layering, branching, and sfx triggering in unity. I went beyond the requirements and designed a B-Game, featuring an awkward physics system where the frog bounces up and down automatically, and the player has to rotate the frog using left and right arrow keys to aim the frog so that it bounces in the desired direction.
