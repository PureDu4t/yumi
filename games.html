I Close My Eyes

Genre: Audio Game/Maze
My Role: Conceptualization, Composer for levels 1, 2, 4, & main theme, Art

I've always dreamed of designing audio games where player location can be precisely told. Over the years, I have come up with a working navigation system using audio and "I Closed My Eyes" was my first attempt in implementing that. The game uses a sfx navigation system, where horizontal and vertical location is represented by one pitch followed by another delayed pitch. The pitches ascend and descent according to the pentatonic scale. This was done to avoid dissonance and make player movement sound like a melody on top of the underscore, which is groove oriented and has no melody. In addition to the location navigation system, we designed the "blind cane," which allows the player to hear the surrounding 8 tiles in clockwise motion. The quality of the tiles are represented with sound effects. For example, if there is a ghost on the tile, the player would hear "ooo." Currently, the game is only functional till the 2nd level, but you can still hear how our complete audio navigation system and "white cane" system works.

Jumpy Frog

Genre: B Game/2D Platformer
My Role: Everything

This game was made for my "Mixing For GAIMS" class at Berklee. I was tasked to create a walkable map that showcases basic implementation of layering, branching, and sfx triggering in unity. I went beyond the requirements and designed a B-Game, featuring an awkward physics system where the frog bounces up and down automatically, and the player has to rotate the frog using left and right arrow keys to aim the frog so that it bounces in the desired direction.
